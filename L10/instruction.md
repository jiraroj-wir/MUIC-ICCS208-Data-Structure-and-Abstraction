# Practice Exercises¶

You will hand in one Java file to Canvas. Choose one of your list implementations from before (SLList, AList, etc.) and make it support the for-each syntax. As we've discussed previously, this will require that you make your class implement Iterable<T>. If you can't make up your mind, we recommend extending your SLList.

While you're at it, make your list implementation properly support .equals as well. A good .equals implementation should not need extra memory, so avoid, for example, dumping eveything into a new array to compare.
